# Sport App 🏋️‍♂️

Мобильное приложение для записи на фитнес-тренировки. Проект демонстрирует навыки создания клиент-серверной логики, валидации данных и работы с локальными базами данных в среде Android Studio.

---

## Основной функционал

*   **Система регистрации и входа:** Многоуровневая проверка данных для обеспечения безопасности и корректности базы пользователей.
*   **Запись на услуги:** Интуитивно понятный интерфейс для выбора и бронирования фитнес-занятий.
*   **История записей:** Отдельный экран (вкладка), где пользователь может просматривать свои активные бронирования, хранящиеся в базе данных.

##  Безопасность и валидация данных

В приложении реализована многоуровневая система защиты и проверки данных, соответствующая современным стандартам разработки:

1.  **Защита от SQL-инъекций:** При работе с базой данных используется **параметризация запросов (подстановка через `?`)**. Это исключает возможность прямой подстановки вредоносного кода в SQL-команды и обеспечивает надежную защиту данных пользователей от взлома.
2.  **Валидация бизнес-логики:**
    *   **Уникальность:** Проверка базы данных на наличие одинаковых логинов при регистрации.
    *   **Формат Email:** Обязательная проверка на соответствие домену `@mail.ru` (использование регулярных выражений или логических фильтров).
    *   **Сложность пароля:** Контроль количества символов для предотвращения использования слабых паролей.

##  Технологический стек

*   **IDE:** Android Studio
*   **Language:** Java / Kotlin 
*   **Database:** SQLite — для хранения учетных записей и истории записей.
*   **UI:** XML (Layouts), Material Design.

##  Архитектура и база данных

Проект включает в себя полноценную работу с данными:
*   Создана структура таблиц для пользователей и записей на тренировки.
*   Реализована логика CRUD (Create, Read), позволяющая сохранять бронирование и отображать его в отдельном списке.
*   Организовано разграничение доступа (различные экраны для авторизованных и неавторизованных пользователей).
